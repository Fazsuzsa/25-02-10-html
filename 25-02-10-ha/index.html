Backup Fragen und Antworten

1. Was sind Backups?

Backups sind Kopien von Daten, die erstellt werden, um diese im Falle von Datenverlust, -beschädigung oder -diebstahl wiederherstellen zu können. Sie dienen als Sicherheitsmaßnahme, um sicherzustellen, dass wichtige Dateien nicht dauerhaft verloren gehen.

2. Warum sollte man Backups machen?

Durch unvorhersehbare Ereignisse wie Hardwarefehler, Softwareprobleme oder menschliche Fehler können Daten verloren gehen. Mit regelmäßigen Backups kann man sicherstellen, dass diese Daten wiederhergestellt werden können.

3. Was macht dieser Code? 

            import shutil
            import datetime

            backup_source = "C:/Users/Benutzer/Dokumente"
            backup_target = "D:/Backup_" + datetime.datetime.now().strftime("%Y%m%d%H%M")

            shutil.copytree(backup_source, backup_target)
            print(f"Backup erfolgreich: {backup_target}")
  1. Der Code importiert die Module shutil und datetime.
  2. Die Quelle für das Backup wird in der Variablen backup_source festgelegt (C:/Users/Benutzer/Dokumente).
  3. Das Ziel des Backups wird in der Variablen backup_target gesetzt. Das Ziel ist ein Ordner auf dem Laufwerk D:, dessen Name das aktuelle Datum und die Uhrzeit enthält.
  4. Der Befehl shutil.copytree() kopiert den gesamten Inhalt des Ordners backup_source in den Ordner backup_target.
  5. Am Ende gibt der Code eine Bestätigung aus, dass das Backup erfolgreich war und zeigt den Zielordner an.

4. Gibt es einen Weg ein Script automatisch wiederholt auszuführen?

Ein Skript kann automatisch wiederholt ausgeführt werden, indem man die Taskplanung in Windows oder Cron-Jobs in Linux verwendet, um das Skript zu festgelegten Zeiten oder in regelmäßigen Intervallen auszuführen.
